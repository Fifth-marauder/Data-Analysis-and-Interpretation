Data Analysis and Interpretation

is a coursera specialisation I'm following. It teaches the basics of data analysis in Python (or SAS), which I found intriguing enough to give it a try. In addition to Python, I'm using R to do (more or less) the same analysis steps and compare the two languages.
The results are actually published and discussed on my tumblr, but are better readable with the GitHub infrastructure behind them. This is why you can find the IPython/Jupyter Notebooks, RMarkdown files and their respective HTML and Markdown result files here. In case you are not familiar with the possibilities, you can nicely display .ipynb files with the nbviewer or HTML files with the HTML preview.

The specialisation consists of four courses: Data Management and Visualization, Data Analysis Tools, Regression Modeling in Practice, Machine Learning for Data Analysis, and a capstone (Data Analysis and Interpretation Capstone).
Below, you can find a list of the courses and their weekly assignments I have already completed and uploaded here:

Data Management and Visualisation (DataManViz)

  • Selecting a research question
  • Writing your first program (Python | R)
  • Managing data (Python | R)
  • Visualising data (Python | R)

Data Analysis Tools (DataAnaT)

  • Hypothesis testing and ANOVA (Python | R)
  • Chi square test of independence (Python | R)
  • Pearson correlation (Python | R)
  • Exploring statistical interactions (Python | R)

Regression Modeling in Practice (RegModPrac)

  • Introduction to regression
  • Basics of linear regression (Python | R)
  • Multiple regression (Python | R)
